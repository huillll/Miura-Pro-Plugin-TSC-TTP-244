# Miura-Pro-Plugin-TSC-TTP-244

一个专为 **TSC TTP-244 系列**热转印标签打印机设计的 Windows WPF 桌面应用，支持在 **30 mm × 100 mm** 标签上可视化编排并打印文本、Code 128 条形码与 QR 二维码。

[![Platform](https://img.shields.io/badge/Platform-Windows-blue.svg)](https://learn.microsoft.com/windows/desktop)
[![Language](https://img.shields.io/badge/Language-C%23-239120.svg?style=flat-square&logo=c-sharp)](https://docs.microsoft.com/dotnet/csharp/)
[![Framework](https://img.shields.io/badge/Framework-.NET_4.7.2-green.svg?style=flat-square)](https://dotnet.microsoft.com/download)

## ✨ 功能特性

- 📝 **标签信息录入**：日期、时间、序列号、产品编号等字段
- 🎨 **可视化编辑**：实时预览画布，所见即所得
- 🖱️ **拖拽定位**：支持自由拖动元素到任意位置（精度 0.1 mm）
- 📐 **缩放调整**：通过右下角手柄调整元素大小
- 💾 **布局持久化**：自动保存自定义布局配置
- 🔧 **打印机配置**：灵活的配置文件管理
- 🖨️ **一键打印**：支持撕纸模式，UTF-8 编码

## 🚀 快速开始

### 系统要求

- Windows 7/10/11（64 位或 32 位）
- .NET Framework 4.7.2 或更高版本
- TSC TTP-244 Plus 系列打印机（USB 连接）

### 安装部署

1. **下载发布包**：从 [Releases](https://github.com/huillll/Miura-Pro-Plugin-TSC-TTP-244/releases) 页面下载最新版本
2. **解压文件**：将整个文件夹解压到任意目录（例如 `C:\Program Files\Miura_Labeling`）
3. **配置打印机**：编辑 `printer.cfg`，将默认打印机名称改为实际使用的 TSC 打印机
   ```
   TSC TTP-244 Plus
   ```
4. **运行程序**：双击 `Miura_Labeling.exe`

### 首次使用

1. 启动程序后，在左侧面板输入标签信息（日期、时间、序列号、产品编号）
2. 在画布上拖拽元素调整位置和大小
3. 点击「打印」按钮即可输出标签

### 标签元素定义

| 元素名称 | 类型 | 默认位置 (mm) | 默认尺寸 (mm) | 说明 |
|---------|------|--------------|--------------|------|
| 日期时间 | 文本 | (1, 2) | 28 × 5 | 显示当前选择的时间 |
| 产品编号 | 文本 | (1, 9) | 28 × 5 | 格式：`PN: {product}` |
| 序列号 | 文本 | (1, 16) | 28 × 5 | 格式：`SN: {serial}` |
| 条形码 | Code 128 | (1, 23) | 28 × 14 | 基于序列号生成 |
| 二维码 | QR Code | (4, 42) | 22 × 22 | 支持变量替换 |

### 操作指南

#### 拖动元素定位

- 将鼠标悬停在画布上的元素上，会出现蓝色边框
- **按住左键拖动**元素到任意位置
- 坐标约束：不可拖出标签范围（30 mm × 100 mm）
- 左侧面板的 X/Y 输入框会实时同步更新

#### 缩放元素大小

- 将鼠标移动到元素的**右下角**，会出现蓝色手柄
- **按住左键拖动**手柄调整宽度和高度
- 最小尺寸：2 mm × 2 mm
- 左侧面板的 W/H 输入框会实时更新

#### 精确位置设置

- 在左侧面板的 X/Y/W/H 输入框中输入数值（单位：mm）
- 支持小数，精度到 0.1 mm
- 修改后画布立即更新

### 打印参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| 标签尺寸 | 30 × 100 mm | 固定尺寸 |
| 间距 (GAP) | 2 mm | 标签间隔 |
| 打印速度 | 4 | 1-12，数值越大越快 |
| 打印浓度 | 12 | 1-30，数值越深越浓 |
| 打印方向 | 纵向 | 支持横向 |
| 字符编码 | UTF-8 | 支持中文 |

### 二维码变量替换

二维码内容支持以下占位符，会自动替换为实际值：

| 占位符 | 替换为 | 示例 |
|--------|--------|------|
| `{serial}` | 序列号 | `SN123456` |
| `{product}` | 产品编号 | `PN-001` |
| `{date}` | 日期 | `2026-06-09` |
| `{time}` | 时间 | `16:26:00` |

**示例配置：**
```
二维码内容 = "{product} - SN{serial} - {date}"
输出结果 = "PN-001 - SN123456 - 2026-06-09"
```

## 🔧 配置文件说明

### printer.cfg

打印机名称配置，每行一个名称：
```
TSC TTP-244 Plus
```

> **提示**：如果找不到你的打印机，请确保已安装官方驱动并在系统中可见。

### layout.cfg

元素布局缓存（自动生成和维护），格式为：
```
日期时间|1.0|2.0|28.0|5.0
产品编号|1.0|9.0|28.0|5.0
序列号|1.0|16.0|28.0|5.0
条形码|1.0|23.0|28.0|14.0
二维码|4.0|42.0|22.0|22.0
```

> **注意**：不建议手动修改此文件，修改后可能导致布局异常。

### 核心逻辑说明

#### 双向联动机制

- 画布拖拽 → 左侧输入框实时更新
- 输入框修改 → 画布元素立即更新
- 防止循环更新：使用 `_busy` 标志位控制

#### 防抖保存

位置参数修改后，延迟 **800 ms** 自动保存到 `layout.cfg`，避免频繁 I/O。

## 🐛 常见问题

### Q: 打印时提示"找不到打印机"

**A:** 
1. 确认打印机已正确连接并开机
2. 在 Windows 设备管理器中检查打印机是否正常显示
3. 修改 `printer.cfg`，填入正确的打印机名称（可从 Windows 设置 → 设备和打印机查看）

### Q: 二维码内容不显示或乱码

**A:** 
1. 确认字符编码为 UTF-8（默认已配置）
2. 检查占位符格式是否正确（如 `{serial}`）
3. 尝试在二维码内容中输入纯英文测试

### Q: 拖拽元素时卡顿或不响应

**A:** 
1. 确保系统性能足够（建议至少 4GB RAM）
2. 如果是虚拟机，启用 GPU 加速
3. 重启程序后重试

### Q: 布局配置丢失

**A:** 
1. 检查 `layout.cfg` 文件是否被删除或损坏
2. 重新调整元素位置后会自动保存
3. 建议定期备份 `layout.cfg`
